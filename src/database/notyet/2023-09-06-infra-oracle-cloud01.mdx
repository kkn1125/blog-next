# 오라클 클라우드

1. 오라클 클라우드
2. 인그레스 규칙 추가
3. 80, 443 포트 추가해도 응답 없음
4. iptables로 별도 설정 필요
5. 테스트 및 확인

```bash
iptables -I INPUT 6 -m state --state NEW -p tcp --dport 443 -j ACCEPT
iptables -I INPUT 6 -m state --state NEW -p tcp --dport 80 -j ACCEPT

netfilter-persistent save
```
